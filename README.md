# Data Analysis Portfolio

A collection of analytical and optimization work built in Microsoft Excel. Each
project folder contains the workbook plus a short write-up of the problem, the
modeling approach, and the result.

> **Note on viewing:** GitHub doesn't preview `.xlsx` files in the browser. To
> explore a workbook, open its folder, read the README for context, then use the
> **Download** button to open the file in Excel (or import it into Google Sheets).

## Skills demonstrated

- **Optimization & prescriptive analytics** — linear, nonlinear, and integer
  programming with Excel **Solver**
- **Decision modeling** — product mix, capital budgeting, pricing, logistics, and
  assignment problems
- **What-if & sensitivity analysis**, data tables, and scenario comparison
- **Financial modeling** — Net Present Value (NPV) and regression-driven decisions

## Projects

### [Optimization Models](optimization-models/)

Operations-research models solved with Excel Solver, spanning the four major
classes of mathematical programming.

| Project | Technique | Example problems |
|---|---|---|
| [Linear Programming](optimization-models/linear-programming/) | Linear optimization | Product mix, budget allocation, investment, NPV |
| [Nonlinear Programming](optimization-models/nonlinear-programming/) | Nonlinear optimization | Price/demand and revenue optimization |
| [Integer Programming](optimization-models/integer-programming/) | Binary / integer optimization | Yes-no capital budgeting, project selection |
| [Transportation & Assignment](optimization-models/transportation-assignment/) | Network optimization | Shipping-cost minimization, staff/resource assignment |
