# Linear Programming

Linear-optimization models solved with Excel **Solver** (Simplex LP engine). The
workbook collects ~20 worked problems, each on its own sheet, where a linear
objective is maximized or minimized subject to linear constraints.

**File:** [`linear-programming.xlsx`](linear-programming.xlsx)

## Representative models

| Problem | Type | Goal |
|---|---|---|
| Budget Model | Allocation | Allocate a fixed budget for best return |
| NCAA T-Shirt Vendor | Product mix | Maximize profit across product lines |
| Woodworks Bookshelf Co. | Product mix | Optimal mix of cherry/oak products under material limits |
| Quality Sweater Company | Production | Maximize margin under labor/material constraints |
| Investment Allocation | Finance | Allocate capital across options for max return |
| Net Present Value / Regression & NPV | Finance | Capital decisions driven by NPV and regression |
| Busing / Bus Company | Logistics | Minimize cost of routing/assignment |
| PC Tech Company, Furniture Company | Production | Profit-maximizing output plans |

## Techniques

- Decision variables, objective cells, and constraint ranges laid out for Solver
- **Simplex LP** engine for guaranteed linear optima
- **What-if analysis** sheet for sensitivity to changing inputs
- Clear separation of *givens*, *decisions*, and *results* on each sheet

## How to view

GitHub can't preview Excel files. **Download** the workbook and open in Excel
(enable Solver via *File → Options → Add-ins*), or import into Google Sheets.
